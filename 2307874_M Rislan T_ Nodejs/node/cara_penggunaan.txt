Cara menggunakan node js
1.Pastikan Node.js sudah terinstal di komputer.
2.Setelah itu buat folder dan file bernama server.js
3.Masuk ke visual code lalu buatlah code codenya, kemudia save
4.Buka terminal atau command prompt.
3.Arahkan ke direktori tempat server.js berada.
 yaitu dengan cara cek versi node.js
	node -v 
 kemudian masuk ke direktori tapi saya telah mencoba di partisi D: tidak bisa, kemudia mencoba 
menggunakan partisi C: untuk menyimpan foldenya akhirnya bisa di akses di prompt
	cd C:\Semester3\Alpro\node
4.Jalankan perintah node server.js.
 maka akan memberikan akses http
5.Buka browser dan akses http://127.0.0.1:3000/ atau http://127.0.0.1:3000/contact.
 nanti akan muncul tulisan "datang......." atau "biodata diri kita", mungkin saja bisa gagal kalau kita ada sedikit saja typo
